This project consists of creating a Python program that generates random passwords.
 The user will be able to specify the length of the password and choose whether to include capital letters,
lowercase letters, numbers and special characters.
This is a simple project that demonstrates the use of functions, user input,
 and generating random characters in Python.
 I think about expanding this project by designing additional features,
  how to save the generated password to a file or embed it in a more application
 complex later.

Este projeto consiste em criar um programa em Python que gera senhas aleatórias.
 O usuário poderá especificar o comprimento da senha e escolher se deseja incluir letras maiúsculas,
letras minúsculas, números e caracteres especiais.
Este é um projeto simples que demonstra o uso de funções, entrada do usuário,
 e geração de caracteres aleatórios em Python.
 Eu penso em expandir este projeto projetando recursos adicionais,
  como salvar a senha gerada em um arquivo ou incorporá-lo em um aplicativo mais 
 complexo posteriormente.
